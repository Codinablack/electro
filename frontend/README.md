# Requirements

<li> NPM </li>

<br>

# Running

<h3> <li> Go to Frontend folder  </li></h3>

```bash
$ cd frontend
```

<h3> <li> Install Dependencies  </li></h3>

```bash
$ yarn install
```

<h3> <li> Run for development  </li></h3>

```bash
$ yarn dev
```

<h3> <li> Run for development  </li></h3>

```bash
$ yarn build
$ yarn start
```

<h3> <li> (Optional) Generate Static Project  </li></h3>

```bash
$ yarn generate
```
